## 随机事件 ### 概念 随机试验E的所有可能结果组成的集合称为E的**样本空间S**。 样本空间的元素，即E的每个结果，称为**样本点**。 试验E的样本空间S的子集为E的**随机事件**，简称**事件**。 当且仅当这一子集中的一个样本点出现时，称这一**事件发生**。 由一个样本点组成的单点集，称为**基本事件**。 全集S在每次试验中总是发生的，称为**必然事件**。 空集∅每次试验中都不发生，称为**不可能事件**。 ### 事件间的关系 包含 \`A sub B\` 相等 \`A sub B\`且\`B sub A\`, \`A = B\` 并（和） \`A uu B\`, \`A + B\` 交（积） \`A nn B\`, \`AB\` 差 \`A - B\` 互斥（互不相容） \`A nn B = O/\` 对立（逆） \`bar A = S - A\` ### 事件的运算 交换律 \`A uu B = B uu A\`; \`A nn B = B nn A\` 结合律 \`A uu (B uu C) = (A uu B) uu C\`; \`A nn (B nn C) = (A nn B) nn C\` 分配律 \`A uu (B nn C) = (A uu B) nn (A uu C)\`; \`A nn (B uu C) = (A nn B) uu (A nn C)\` 对偶律 \`bar (A uu B) = bar A nn bar B\`; \`bar (A nn B) = bar A uu bar B\` 吸收律 \`A uu S = S\`; \`A nn S = A\`; \`A uu O/ = A\`; \`A nn O/ = O/\` 重余律 \`bar bar A=A\` 幂等律 \`A uu A = A\`; \`A nn A = A\` 差化积 \`A - B = A nn bar B\` ## 概率 ### 定义 对于E的每一事件赋予一个实数，记为P(A)，称为事件A的**概率**，如果集合函数P满足下列条件: * **非负性**：对于每一事件A，有\`P(A) >= 0\` * **归一性**：\`P(S) = 1\` * **可列可加性**：设是\`A_1, A_2, ...\`是两两互不相容的事件，则有\`P(A_1 uu A_2 uu ...) = P(A_1) + P(A_2) + ...\` ### 性质 * \`P(O/) = 0\` * **有限可加性**：设是\`A_1, A_2, ..., A_n\`是两两互不相容的事件，则有\`P(A_1 uu A_2 uu ... uu A_n) = P(A_1) + P(A_2) + ... + P(A_n)\` * 若\`A sub B\`，则有\`P(B-A) = P(B) - P(A)\`; \`P(B) >= P(A)\` * \`P(A) <= 1\` * \`P(bar A) = 1 - P(A)\` * \`P(A uu B) = P(A) + P(B) - P(A nn B) = P(A) + P(B) - P(AB)\` \`P(A\_1 uu A\_2 uu ... uu A\_n) = sum\_(i=1)^n P(A\_i) - sum\_(1<=i<=j<=n)P(A\_iA\_j) + sum\_(1<=i<=j<=k) P(A\_iA\_jA\_k) + ... + (-1)^(n-1) P(A _1A\_2...A\_n) \` ## 等可能概型（古典概型） 每个基本事件发生的可能性相同：\`P(A) = k/n = (A包含的基本事件数)/(S中基本事件的总数)\` **实际推断原理**: 概率很小的事件在一次实验中实际上几乎是不发生的。 ### 排列组合 \`P_n^k = (n!)/((n-k)!)\`; 若k可以重复出现（放回抽样），则 \`U_n^k = n^k\` \`C\_n^k = P\_n^k/(k!) = (n!)/(k!(n-k)!)\`；若k可以重复出现，则 \`H\_n^k = C\_(n+k-1)^k\` \`C_n^k = C_n^(n-k)\` \`C\_n^k = (n/k)C\_(n-1)^(k-1)\` \`C\_n^k = C\_n^(k-1) + C\_(n-1)^(k-1)\` ## 条件概率 事件A发生的条件下事件B发生的**条件概率**：\`P(B|A) = (P(AB))/(P(A))\` * 对于每一事件B，有\`P(B|A) >= 0\` * \`P(S|A) = 1\` * 设是\`B\_1, B\_2, ...\`是两两互不相容的事件，则有\`P(uuu\_(i=1)^oo B\_i|A) = sum\_(i=1)^oo P(B\_i|A)\` ### 乘法公式 \`P(AB) = P(B|A)P(A) = P(A|B)P(B)\` \`P(A\_1A\_2...A\_n) = P(A\_n|A\_1A\_2...A\_(n-1))P(A\_(n-1)|A\_1A\_2...A\_(n-2))...P(A\_2|A\_1)P(A\_1)\` ### 全概率公式 若\`B\_1, B\_2, ..., B\_n\`为S的一个划分，则 \`P(A) = P(A|B\_1)P(B\_1) + P(A|B\_2)P(B\_2) + ... + P(A|B\_n)P(B\_n)\` 若\`B_i\`还可以继续划分，则可对\`B_i\`递归使用全概率公式。 e.g: 求解[蒙提霍尔问题](http://zh.wikipedia.org/wiki/%E8%92%99%E6%8F%90%E9%9C%8D%E7%88%BE%E5%95%8F%E9%A1%8C) ``` +-------------------+------------------+-----------------+-----------------+ | P(B) | P(C|B) | P(A|CB)坚持选择 | P(A|CB)改变选择 | +-------------------+------------------+-----------------+-----------------+ | 参赛者挑羊(2/3) | 主持人挑羊(1) | 中奖(0) | 中奖(0) | +-------------------+------------------+-----------------+-----------------+ | 参赛者挑汽车(1/3) | 主持人挑A羊(1/2) | 中奖(1) | 中奖(0) | | |------------------+-----------------+-----------------+ | | 主持人挑B羊(1/2) | 中奖(1) | 中奖(0) | +-------------------+------------------+-----------------+-----------------+ ``` 坚持选择: \`2/3 xx 1 xx 0 + 1/3 xx 1/2 xx 1 + 1/3 xx 1/2 xx 1 = 1/3\` 改变选择: \`2/3 xx 1 xx 1 + 1/3 xx 1/2 xx 0 + 1/3 xx 1/2 xx 0 = 2/3\` ### 贝叶斯(Bayes)公式 若\`B\_1, B\_2, ..., B\_n\`为S的一个划分，则 \`P(B\_i|A) = (P(B\_iA))/(P(A)) = (P(A|B\_i)P(B\_i))/(sum\_(j=1)^n P(A|B\_j)P(B\_j))\` ## 期望 Expectation 以概率为权重的加权平均值。 \`\text{E}[X] = \sum_{x \in \mathcal{X}}xP(x)\` ## 方差 Variance 一个随机变量与它的期望之间的差的平方的加权平均值。 \`\text{Var}(X) = \text{E}[(X - \text{E}[X])^2]\` ## 独立性 事件A, B为相互独立的事件：\`P(AB) = P(A|B)P(B) = P(A)P(B)\` * 若事件A，B相互独立，则\`A与bar B\`， \`bar A与B\`， \`bar A与bar B\`也相互独立。 * 若\`P(A)>0, P(B)>0\`，事件A，B相互独立与A，B互不相容不能同时成立。 * 必然事件和不可能事件与任何事件都独立。 * \`P(A uu B) = 1 - P(bar A bar B) = 1 - P(bar A)P(bar B)\` 在实际应用中，对于事件的独立性，往往不是根据定义来判断，而是根据实际意义来加以判断的。 ##
